# Contract

A legally enforceable agreement document — NDAs, MSAs, SOWs, employment agreements, vendor agreements, and licensing agreements. Both executed contracts and reusable templates with bracketed placeholders are recognized, so the same library surface holds your signed paper and the forms you reuse.

## Capabilities

- Hold the full body of a signed contract or reusable template
- Search and retrieve a past agreement when reviewing similar terms
- Attach as reference context when an agent needs grounded legal language
- Compare contracts side by side when negotiating new versions
